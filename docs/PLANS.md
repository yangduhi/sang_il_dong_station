# Plans

Use `plans/` for stage-scoped work. Each stage should have:

- a short plan created before edits
- a result report after verification
- open risks and blockers

Templates:

- [`plans/templates/stage-plan-template.md`](../plans/templates/stage-plan-template.md)
- [`plans/templates/stage-report-template.md`](../plans/templates/stage-report-template.md)
